💾 สำรอง / กู้คืนข้อมูล

ข้อมูลเก็บใน browser ของเครื่องนี้เท่านั้น แนะนำให้ Export เก็บไว้สม่ำเสมอ

วิธีใช้

1. แท็บ บันทึก → พิมพ์ชื่อหรือเบอร์คนกิน เลือกจากรายการ หรือเพิ่มใหม่
2. ลูกค้าใหม่ — กรอกแค่ชื่อก็พอ (เบอร์เป็น optional)
3. มีตัวแทนสั่ง → ติ๊ก toggle เพื่อระบุคนสั่งแทน — สิทธิ์ตกที่คนกินเสมอ
4. แท็บ เช็ก/แลก ค้นหาด้วยชื่อ/เบอร์ แล้วกดแลกฟรี
5. Export JSON เก็บไว้ใน Drive/iCloud เป็นประจำ
